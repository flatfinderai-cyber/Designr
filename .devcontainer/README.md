# Designr Automation System - Codespaces Configuration

This devcontainer configuration provides a ready-to-use environment for the **Designr Automation System**.

## What Gets Configured

- **Node.js 18** - For running JavaScript automation tools
- **Python 3.11** - For Python-based automation scripts
- **Git** - Version control
- **VS Code Extensions** - ESLint, Prettier, Python support

## Quick Start with Codespaces

1. Click the "Code" button on GitHub
2. Select "Codespaces" tab
3. Click "Create codespace on [branch]"
4. Wait for the environment to build (~2-3 minutes)
5. Your terminal will open in the `/workspaces/Designr` directory
6. Navigate to `automation/` directory to start working

## What's Included

The automation system includes:
- **Scripts** - Helper scripts for setup and verification
- **Templates** - Setup templates for common tools
- **Workflows** - Example automation workflows
- **Documentation** - Complete guides on using the system
